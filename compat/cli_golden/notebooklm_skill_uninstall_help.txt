Usage: notebooklm skill uninstall [OPTIONS]

  Remove the NotebookLM skill from supported agent directories.

Options:
  --scope [user|project]        Remove user-level or project-level skill
                                installs.  [default: user]
  --target [all|claude|agents]  Remove Claude Code, universal agent skill
                                directories, or both.  [default: all]
  --help                        Show this message and exit.
