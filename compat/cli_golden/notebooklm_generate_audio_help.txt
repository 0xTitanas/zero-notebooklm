Usage: notebooklm generate audio [OPTIONS] [DESCRIPTION]

  Generate audio overview (podcast).

  Use --json for machine-readable output.

  Example:
    notebooklm generate audio "deep dive focusing on key themes"
    notebooklm generate audio "make it funny and casual" --format debate
    notebooklm generate audio -s src_001 -s src_002 "from specific sources"

Options:
  --prompt-file PROMPT_FILE       Read prompt/query text from a file (or '-'
                                  for stdin) instead of the positional
                                  argument
  -n, --notebook TEXT             Notebook ID (uses current if not set).
                                  Supports partial IDs.  [env var:
                                  NOTEBOOKLM_NOTEBOOK]
  --format [deep-dive|brief|critique|debate]
                                  Conversation style (default: deep-dive)
  --length [short|default|long]   Audio length: short, default, or long
  --language TEXT                 Output language (default: --language >
                                  NOTEBOOKLM_HL env > config > 'en')
  -s, --source TEXT               Limit to specific source IDs
  --wait / --no-wait              Wait for completion (default: no-wait)
  --timeout INTEGER               Maximum seconds to wait (default: 1200)
  --interval INTEGER RANGE        Seconds between status checks (default: 2)
                                  [x>=1]
  --retry INTEGER RANGE           Retry N times with exponential backoff on
                                  rate limit  [x>=0]
  --json                          Output as JSON
  --help                          Show this message and exit.
