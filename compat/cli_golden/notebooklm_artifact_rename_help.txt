Usage: notebooklm artifact rename [OPTIONS] ARTIFACT_ID NEW_TITLE

  Rename an artifact.

  ARTIFACT_ID can be a full UUID or a partial prefix (e.g., 'abc' matches
  'abc123...').

Options:
  -n, --notebook TEXT  Notebook ID (uses current if not set). Supports partial
                       IDs.  [env var: NOTEBOOKLM_NOTEBOOK]
  --json               Output as JSON
  --help               Show this message and exit.
