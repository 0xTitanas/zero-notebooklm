Usage: notebooklm source add-drive [OPTIONS] FILE_ID TITLE

  Add a Google Drive document as a source.

Options:
  -n, --notebook TEXT             Notebook ID (uses current if not set).
                                  Supports partial IDs.  [env var:
                                  NOTEBOOKLM_NOTEBOOK]
  --mime-type [google-doc|google-slides|google-sheets|pdf]
                                  Document type (default: google-doc)
  --json                          Output as JSON
  --help                          Show this message and exit.
