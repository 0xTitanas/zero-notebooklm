Usage: notebooklm list [OPTIONS]

  List all notebooks.

  Pagination & display:
    --limit N         Show at most N notebooks (default: unlimited).
    --no-truncate     Do not truncate the Title column in the table view.

Options:
  --json                 Output as JSON
  --limit INTEGER RANGE  Show at most N rows. 0 = show no rows. Omit for
                         unlimited. Applies to both text and --json output.
                         [x>=0]
  --no-truncate          Disable column truncation in the rendered table
                         (default: truncate).
  --help                 Show this message and exit.
