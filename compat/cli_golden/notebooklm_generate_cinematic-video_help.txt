Usage: notebooklm generate cinematic-video [OPTIONS] [DESCRIPTION]

  Generate cinematic video overview (AI-generated documentary footage).

  Alias for 'generate video --format cinematic'. Uses Veo 3 AI to create
  documentary-style videos. Requires Google AI Ultra.

  Note: --format is locked to 'cinematic' on this subcommand; passing any
  other value (e.g. --format explainer) raises an error. Use 'generate video
  --format <other>' for non-cinematic formats.

  Example:   notebooklm generate cinematic-video "documentary about quantum
  physics"

Options:
  --prompt-file PROMPT_FILE       Read prompt/query text from a file (or '-'
                                  for stdin) instead of the positional
                                  argument
  -n, --notebook TEXT             Notebook ID (uses current if not set).
                                  Supports partial IDs.  [env var:
                                  NOTEBOOKLM_NOTEBOOK]
  --format [explainer|brief|cinematic]
                                  Video format; 'cinematic' uses Veo 3 footage
                                  (default: explainer)
  --style [auto|custom|classic|whiteboard|kawaii|anime|watercolor|retro-print|heritage|paper-craft]
                                  Visual style (default: auto). Use 'custom'
                                  with --style-prompt.
  --style-prompt TEXT             Custom visual style prompt
  --language TEXT                 Output language (default: --language >
                                  NOTEBOOKLM_HL env > config > 'en')
  -s, --source TEXT               Limit to specific source IDs
  --wait / --no-wait              Wait for completion (default: no-wait)
  --timeout INTEGER               Maximum seconds to wait (default: 1800;
                                  cinematic format defaults to 3600)
  --interval INTEGER RANGE        Seconds between status checks (default: 2)
                                  [x>=1]
  --retry INTEGER RANGE           Retry N times with exponential backoff on
                                  rate limit  [x>=0]
  --json                          Output as JSON
  --help                          Show this message and exit.
